Overview

A LaTeX-enabled flashcard application with spaced repetition.

It is open source; the source code is available [a href="https://github.com/adrianvla/decks"]here[/a].

You can use it online on [btn link="https://decks.morisinc.net/"]decks.morisinc.net[/btn]. If you have trouble, please visit the [a href="https://decks.morisinc.net/help"]Help page[/a].

To set it up, you'll have to create a database that can be used across many Morisinc services as explained [a href="https://decks.morisinc.net/help"]here[/a].

Features

  • LaTeX Support
  • Normal flashcard mode (like Quizlet)
  • Self-hosted database (requiring each user to enter their own firebase api-key)
  • Cross-platform (online WebApp)
  • Flashcard Deck Import & Export
  • Dark & Light mode

Why I built this

I previously used [a href="https://www.studysmarter.de/"]StudySmarter[/a] to prepare for exams, but encountered instability when creating flashcards rapidly.

Two days before my Calculus I (Part 1) exam, a significant portion of my flashcards disappeared due to platform issues.

In response, I built the first prototype of this app in one week and used it for Part 2 of the exam.

Decks prioritizes stability and fast flashcard creation to protect your work.
