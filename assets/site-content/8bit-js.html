View the live demo [btn link="https://adrianvla.github.io/8BIT-JS-Example/"]here[/btn].

The engine uses the HTML5 Canvas API and sprite sheets. It supports infinite scrolling, parallax effects, and particle systems.

The project repository and documentation are available [a href="https://github.com/adrianvla/8BIT-JS"]here[/a]. Have fun creating retro or modern games!

Documentation is provided in the repository's README.md. The physics module is still a work in progress. The design is reminiscent of Scratch—using functions such as put() and stamp()—but is implemented in JavaScript for performance and flexibility.
