Installation

  • Download the repo
  • Replace M5.BtnA.isPressed() at line 112 of src/main.cpp to match your tube's pulse signal for radiation counting.
if (M5.BtnA.isPressed()) {
    if (flag_tick) {
      signalHandler();

      flag_tick = false;
    }
    } else {
    flag_tick = true;
    }
  • The 2nd button (button B) resets the counter
  • Compile the project for M5Stack Core. I used PlatformIO in Visual Studio Code instead of the Arduino IDE.
  • Program the M5 stack with the compiled code
  • Turn on the M5Stack

How To use

  • Turn on M5Stack by pressing the red button.
  • Connect your device to the M5Stack_setup Wi-Fi network and follow the instructions on the M5Stack screen.
  • Go to http://192.168.4.1 on your device
  • You're done!
