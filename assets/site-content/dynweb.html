How to Use

Set up a folder named a alongside dynweb.js, and place your HTML files there (for example, home.html or hello.html). Create an HTML page for your website and include an element with the id content (a dedicated tag is recommended). When a user clicks a link or visits a URL like https://www.example.com/#something, the element with id content will be populated with the contents of a/something.html.
