Overview

Quick! is an online web-app that lets you plan everything, from small to-do tasks to huge projects, by the minute or even by the second.

The app is fully open source; you can find its source code [a href="https://github.com/adrianvla/quick"]here[/a].

You can use it on [a href="https://quick.morisinc.net/"]quick.morisinc.net[/a].

To set it up, you'll have to create a database that can be used across many Morisinc services as explained [a href="https://decks.morisinc.net/help"]here[/a].
