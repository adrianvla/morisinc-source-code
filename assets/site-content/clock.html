Overview

A matrix clock built on a Raspberry Pi Zero, similar in spirit to the LaMetric Time.
Work in progress.

You can find the source code and build files [a href="https://github.com/adrianvla/clock"]here[/a].

Features

Features include movable widgets, multiple screen environments, and animations.
A control panel allows you to edit nearly everything, including widget positions and widget code.

Clock drift on the Raspberry Pi Zero is mitigated by periodic synchronization with an online time server.

Widgets

Widgets that are provided out-of-the-box:

  • Days of Week (with bars that look like this if you're on a Saturday: ░░ ░░ ░░ ░░ ░░ ▓▓ ░░)
  • Calendar (indicates the current date with a nice calendar-like background)
  • Digital Time: 3x5 font, HH : MM (: SS, configurable)
  • Big Digital Time: 5x5 font, HH : MM
  • Rss Feed: Scrolls news headlines on the display, 8x8 font, supports Latin, Cyrillic, Japanese Kana&Kanji
  • Temperature: temperature°C
  • Weather: A nice 8x8 widget with animated icons for every weather type, with even a particle system for rain

Screens

Screens are analogous to desktops on Windows or macOS. Widgets are placed on screens, and based on time or events (for example, receiving an RSS update), the screens cycle with smooth animations.

Control panel

With the control panel, you can:

  • Change the password
  • View what's on the display
  • Change the brightness
  • Restart
  • Edit Widgets / Create new ones using JavaScript right in your browser (using Monaco Editor)
  • Edit Screens / Create new ones by dragging and dropping the widgets on a virtual screen

How to Use

Obtain an API key from weatherapi.com.
Paste it into /code/src/widgets/weather/weather.js (work in progress; will be moved to the control panel).

Configure your RSS feeds in /code/src/widgets/rssFeed/rss.js (work in progress; will also be moved to the control panel).

Create a systemctl service to autorun

#!/bin/bash
cd <the directory containing the contents of the code folder>
sudo node src/main.js

and name it script-startup.service

And you're good to go.

To connect the WS281x display, use GPIO 18, or change the pin in /code/src/utils/driver.js.
