I built this recreation from scratch using my own Tetris engine, which supports different tetromino colors. The user interface mirrors the original.

How to play

Controls:

  • 1: Show next tetromino
  • 5 / Space: Drop
  • 7 / Left arrow: Move left
  • 8 / Up arrow: Rotate
  • 9 / Right arrow: Move right
