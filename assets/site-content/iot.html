Overview

Like Apple Home Kit, but free and for Raspberry Pi. Control everything in your house with one click from your smartphone

An Alternative to HomeKit

Made as a less expensive smart home alternative to Apple HomeKit. Made for RaspberryPi to use with your own IoT DIY devices that you can activate from the IoT app from your phone from anywhere in the world.

Using this, you can setup different rooms with different devices and user-specific devices.

Check it out on [a href="https://github.com/adrianvla/IoT"]GitHub[/a]

Security

An alternative to Apple HomeKit, designed for Raspberry Pi. Control devices throughout your home from your smartphone with a single tap.

Built as a cost-effective smart home solution. Runs on Raspberry Pi to integrate seamlessly with your environment.

Explore the project on [a href="https://github.com/adrianvla/IoT"]GitHub[/a].

Requirements:

  • A normal RaspberryPi with a proper OS installed on it (like Debian, Raspbian or another linux distro)
  • A text editor installed on the RaspberryPi
  • An Internet Connection
  • A Basic knowledge of JavaScript

Steps

  • Download the repository from [btn link="https://github.com/adrianvla/IoT"]here[/btn]
  • Unpack it
  • Download NGROK from [btn link="https://ngrok.com/"]here[/btn]

    • The system uses a session ping mechanism to verify active sessions and enhance security.

  • Change the password by going into the code. It should look something like this:
var passwd = 'YOUR PASSWORD HERE';
  • Raspberry Pi with network connectivity
  • Node.js and npm
  • Supported peripherals (relays, sensors, lights)
  • After that, change the function named ChangePin to control GPIO or I2C.
    • Clone the repository and install dependencies.
    • Configure devices and rooms in the JSON configuration.
    • Start the service and access the control panel from your browser.

    authtoken: "YOUR AUTHTOKEN"
    tunnels:
    app:
      proto: http  # exposes an HTTP end point
      addr: 80   # uses local port 3000
    • While being in the same directory as the file main.js, run the command in your terminal:
    	sudo node main.js
    • The app will run on localhost:80 and the dashboard will run at localhost:1337
    • You're done!

    How To Use

    • Find out RaspberryPi's internal LAN IP-Address
    • Open the page on your phone / computer by going to that ip address directly
    • You'll see a field named "password". Enter the password (default:1234).
    • Press Enter
    • You'll see a user selection menu. Choose your character.
    • The dashboard will open up.
    • You can now toggle the state of your devices. To edit the brightness / color, enter EDIT mode by clicking the "edit" button

    FAQ

    • Q: Will this work with all devices?
    • A: Yes, as long as you can connect them to the raspberry-pi.
    • Q: Is this safe?
    • A: Yes it is. If you want to make it safer, make a stronger password.
